## 🖥️ Projeto 
Este é um projeto web responsivo de uma cápsula do tempo para exibir memórias em uma timeline.

## 🚀 Tecnologias
Projeto desenvolvido durante o NLW da Rocketseat usando as seguintes tecnologias:

-HTML
-CSS
-GIT
-GITHUB
